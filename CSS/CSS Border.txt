CSS Border:-
==========

CSS provide border attribute that allows to define border for sspecified elements it includes 

1)border-style
2) border-color
3) border-width


boder-style:-
============

CSS provides predefined element for specified element which include groove, double-dotted, dassh etc.

You can apply bordeer style individually for the elements  with left, right, top, bottom presets

Syntax:-
=======
  div{
       border-left-style:double;
       border-right-style:dotted;
       border-top-style:dashed;
       border-bottom-style:solid;
     }

border-color:-
============

It applies a color for border which you can individually define for presets

Syntax:-
=======

div{
      border-left-color:red;
      border-right-color:green;
      border-top-color:blue;
      border-bottom-color:yellow;
    }

border-width pixels:-
===================

div{
       border-left-width:10px;
       border-right-width:10px;
       border-top-width:5px;
       border-bottom-width:5px;
   }
